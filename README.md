# APK.TW 自動簽到 Chrome 擴展

一個智能化的 Chrome 擴展，專為 APK.TW 論壇設計的自動簽到工具。

## ✨ 功能特色

- 🤖 **智能自動簽到** - 每日定時自動執行簽到
- 🎯 **狀態智能檢測** - 自動識別登入和簽到狀態  
- 📊 **實時狀態監控** - 直觀顯示簽到結果和歷史記錄
- 🔔 **通知提醒系統** - 簽到成功及時通知
- ⚙️ **靈活設置選項** - 自定義簽到時間和通知偏好
- 🌈 **美觀用戶界面** - 現代化設計，操作簡單直觀

## 🚀 快速開始

### 安裝步驟

1. **下載擴展文件**
   ```bash
   git clone [repository-url]
   cd apk-tw-auto-signin
   ```

2. **打開 Chrome 擴展管理頁面**
   - 在地址欄輸入 `chrome://extensions/`
   - 開啟「開發者模式」

3. **載入擴展**
   - 點擊「載入已解壓縮的擴展程式」
   - 選擇擴展文件夾

4. **首次設置**
   - 打開 APK.TW 並登入帳號
   - 點擊擴展圖標進行初始設置

### 使用方法

1. **檢查狀態**
   - 點擊擴展圖標
   - 點擊「檢查狀態」按鈕

2. **手動簽到**
   - 確保已登入 APK.TW
   - 點擊「手動簽到」按鈕

3. **設置自動簽到**
   - 開啟「自動簽到」選項
   - 設置簽到時間（建議 09:00）
   - 開啟「通知提醒」

```
│   └── icon128.png
└── README.md              # 說明文檔
```

### 核心技術

- **Manifest V3** - 使用最新的 Chrome 擴展 API
- **Service Worker** - 背景腳本處理定時任務
- **Content Script** - 頁面腳本處理 DOM 操作
- **Chrome Storage API** - 本地數據存儲
- **Chrome Alarms API** - 定時任務管理

### 簽到機制

1. **登入檢測** - 檢查用戶是否已登入 APK.TW
2. **簽到狀態檢測** - 識別簽到按鈕和已簽到狀態
3. **自動簽到執行** - 模擬點擊簽到按鈕
4. **結果驗證** - 確認簽到是否成功
5. **日誌記錄** - 保存簽到歷史記錄

## ⚙️ 設置選項

| 選項 | 說明 | 預設值 |
|------|------|--------|
| 自動簽到 | 每日定時自動執行簽到 | 開啟 |
| 簽到時間 | 自動簽到的執行時間 | 09:00 |
| 通知提醒 | 簽到成功時顯示通知 | 開啟 |

## 🐛 故障排除

### 常見問題

**Q: 簽到失敗，提示「未登入帳號」**
A: 請確保您已經在 APK.TW 網站登入帳號

**Q: 找不到簽到按鈕**
A: 請訪問論壇頁面（如 forum-soft-1.html）

**Q: 自動簽到沒有執行**
A: 檢查設置中的自動簽到選項是否開啟

**Q: 擴展無法正常載入**
A: 請確認 Chrome 瀏覽器版本支援 Manifest V3
